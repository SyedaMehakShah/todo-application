# Frontend Skill: Build Pages & Components

## Instructions
Create frontend pages and components with these features:

1. **Page & Layout Structure**
   - Semantic HTML structure
   - Header, footer, main content sections
   - Responsive grid or flex layouts

2. **Components**
   - Reusable UI components (buttons, cards, modals, forms)
   - Props-driven or configurable design
   - Consistent spacing and alignment

3. **Styling**
   - Use modern CSS techniques: Flexbox, Grid
   - Apply design patterns like glassmorphism, neumorphism, or material design
   - Responsive typography, spacing, and color scheme

4. **Interactivity**
   - Hover effects, transitions, and animations
   - Form validation and interactive elements
   - Mobile-friendly touch interactions

5. **Responsiveness**
   - Works across all screen sizes
   - Uses relative units (%, rem, vh/vw)
   - Touch-friendly elements for mobile devices
